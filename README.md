# 🎓 Smart Classroom Analytics System

An AI-powered system that helps identify students who may be at academic risk by analyzing attendance, assignment scores, quiz performance, and participation data. The goal is to provide early insights and recommendations to improve student outcomes.

---

## 📌 Problem Statement

In many educational institutions, students who are struggling academically are often identified only after major exams. This project aims to predict student performance early and provide actionable recommendations to help students improve before it is too late.

---

## 🚀 Features

- Predict student academic performance
- Identify at-risk students
- Generate risk levels (Low, Medium, High)
- Visualize student performance trends
- Analyze attendance and assessment data
- Provide personalized improvement suggestions
- Interactive web dashboard
